# DOOMFire

Implementation of DOOM fire algorithm

## Quick start

```sh
go build
./DOOMFire
```

| Key          | Usage                  |
| ------------ | ---------------------- |
| <kbd>q</kbd> | Increase fire intesity |
| <kbd>e</kbd> | Decrease fire intesity |
